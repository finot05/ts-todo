## Features
- Add tasks
- Edit tasks (using a pen icon)
- Delete tasks (using an X icon)
- Responsive and clean UI

## Built Using
- TypeScript
- HTML & CSS
- Font Awesome (for icons)

## How to Run
1. Clone or download the repo
3. Run `npm install`
